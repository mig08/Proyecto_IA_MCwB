Como ejecutar archivos:

MCwB.cpp: Se compila utilizando el comando "make" y luego se ejecuta con ./mcwb. Es necesario que
la instancia este en la misma carpeta que MCwB.cpp. Luego es posible limpiar los ejecutables utilizando
el comando "make clean". 

tablas_y_graficos_mcwb.ipynb: Para que genere las tablas y los graficos es necesario que esten los siguientes .csv en la misma carpeta: routes_greedy.csv, routes_sa.csv, sa_iterations.csv, summary.csv. Estos archivos son generados despues de ejecutar MCwB.cpp.
